분할매수 : 전일종가 or 시초가 대비 +/-ㅜ%구간에서 나누어 매수하는 방식입니다. 백테스팅,가상매매,실전매매 모두 적용됩니다.
운용자금 : 초기 투자금액을 입력
운용기간 : 백테스팅을 실행할 기간을 입력
수수료율 : 사용하는 증권사의 매매 수수료율 입력
슬리피지 : 실거래에서 호가공백, 동시호가로 인해 발생하는 주문가와 체결가의 차이. 슬리피지를 입력하면 실제 수익률과 더욱 유사해질 가능성이 높음 (입력하지 않아도 무방)
매수 가격기준 : 매수할 종목을 어느 가격에 매수할 것인지 전일 종가를 기준으로 입력
종목당 매수비중 : 한 종목을 총 자산의 몇%만 매수할 것인지 입력 (매수비중이 10%, 총 자산이 1000만원이라면 한 종목당 100만원씩 매수)
종목당 최대 매수금액 : 한 종목당 최대로 매수할 수 있는 금액을 설정 (백테스팅 기간이 길수록 복리로 인해 자산규모가 늘어나기 때문에, 최대 매수금액을 설정하면 실제 수익률과 유사해질 가능성이 높음)
최대 보유종목 수 : 최대로 보유할 수 있는 종목의 수 입력 (매수비중이 10%, 최대 보유종목 수가 5종목이면 종목당 10%식 5종목을 매수하고 총자산의 50%는 현금 보유)
1일 최대 매매종목 수 : 하루에 최대로 매수할 수 있는 종목의 수 입력 (3종목 제한일 경우, 매수예정 종목이 5종목 이더라도 상위 3개의 종목만 매수)
필터링 조건 : 관리종목과 감리종목을 제외, 포함 할지를 결정
재매수 방지 조건 : 재매수 방지를 1일로 선택 할 경우 매도한 종목이 다음 날 추천 종목으로 나오지 않게됨
퇴출(stop) 조건 : 매수한 종목의 목표가와 손절가를 입력
종목 최소 보유일 : 3일을 입력하면, 매수 후 3영업일간 목표, 손절가가 오더라도 매도를 하지 않음
종목 최대 보유일 : 5일을 입력하면, 매수 후 5영업일 뒤에 매도를 함
만기매도 가격기준 : 최대 보유일을 만족하여 매도할 경우의 매도가격을 입력
조건 부합시 매도 가격기준 : 매도조건에 의해 매도할 경우의 매도가격을 입력
트레일링스탑 : 전고점 대비 설정한 손절가 밑으로 가격이 내려가면 매도
종가청산기능 : 종가 청산 기능을 사용할 경우 : 만기매도/조건매도일 보유 종목 매도가 전량 이루어지지 않았을 경우 종가에 모두 매도(청산) 종가청산 기능을 사용하지 않을 경우: 만기매도/조건 매도일 보유종목 매도가 전량 이루어지지 않으면 다음날에 다시 매도를 시도
